主要功能

🚫

屏蔽不想看到的网站

一键屏蔽垃圾站点和内容农场。立即清理您的搜索结果。

⭐

高亮可信来源

标记并高亮您信任的网站。不错过优质内容。

🌐

全平台支持

支持 15+ 主流搜索引擎，包括 Google、Bing、DuckDuckGo、Yahoo、Yandex、Startpage、Ecosia、Ask、AOL、Naver、Brave Search、OneSearch、SearX、Qwant等。提供一致的使用体验。

🔒

注重隐私

所有数据存储在本地。无跟踪，无分析，完全私密。

⚡

极速响应

零性能影响。浏览搜索结果时即时生效。

📱

跨浏览器支持

支持 Chrome、Firefox、Edge 等主流浏览器。

快速开始

1️⃣

从浏览器商店安装 ZenSearch

Chrome Edge Firefox
2️⃣

点击工具栏中的 ZenSearch 图标

安装完成后，您会在浏览器工具栏看到 ZenSearch 的图标

3️⃣

开始屏蔽或高亮网站

在搜索结果页面上：

  • 右键链接，菜单项中选择高亮或屏蔽网站
  • 快捷键Alt+H快速屏蔽鼠标悬停的网站
